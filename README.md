# CyberAggregator

The Cyber Aggregator is a smart Smartphone Comparison Dashboard. It acts as a specialized search and evaluation hub that stores a massive, up-to-date database of mobile phones (ranging from budget champions like the OnePlus Nord CE 6 Lite to ultra-flagships like the Samsung Galaxy S26 Ultra and iPhone 17 Pro Max).

Its main goal is to let users filter through device parameters and run a "verdict engine" to instantly find out which phone offers the absolute best hardware value for money.
